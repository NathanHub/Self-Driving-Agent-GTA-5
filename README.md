# Self-Driving-Agent-GTA-5
RUN this code on NVIDIA 1060 and above gpu with tensorflow-gpu installed

for this project to work 
you will need following

Non python requirements

1. Vjoy
2. Xbox Vbus for Vjoy
3. open iv
4. GTA 5

Python requirements

1. Python 3.6 and above (IMPORTANT)
2. opencv
3. tensorflow-gpu
4. darkflow "https://github.com/thtrieu/darkflow"
5. keras
6. cython
7. shapely
8. pywin32 (shapely and pywin32 can be found in "https://www.lfd.uci.edu/~gohlke/pythonlibs/")

Important steps required

after installing all above mentioned restart computer because some of the pythin libraries requires a restart to work.
